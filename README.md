# 🤖 Бот для канала @AiContentCreatorUZ

Этот бот публикует посты на узбекском языке каждый день в 10:00 по Ташкенту.

## 📋 Что нужно перед запуском

1. **Токен бота** — уже есть ✅
2. **API-ключ Claude** — получить на console.anthropic.com (платно, ~5$ хватит на 6 месяцев)
3. **Ключ Unsplash** — бесплатно на unsplash.com/developers

## 🚀 Где запускать бота

Самый простой вариант — **Railway.app** (бесплатно первые 5$ кредита, хватит на 1-2 месяца):

1. Регистрируешься на railway.app
2. Создаёшь новый проект → Deploy from GitHub
3. Загружаешь файлы бота
4. В переменных окружения добавляешь:
   - `BOT_TOKEN` = твой токен
   - `ANTHROPIC_API_KEY` = ключ Claude
   - `UNSPLASH_ACCESS_KEY` = ключ Unsplash
5. Готово! Бот работает 24/7

## 📅 Расписание постов

- Понедельник, среда, пятница — **промпты для AI**
- Вторник — **новости AI**
- Четверг — **советы и лайфхаки**
- Суббота — **кейсы**
- Воскресенье — **мотивация**

Все посты на узбекском, с картинкой с Unsplash.

## ⚙️ Как изменить время или темы

Открой `bot.py`:
- Строка `POST_HOUR = 10` — час публикации
- Словарь `POST_TYPES` — какие типы постов в какие дни
- Словарь в функции `generate_post` — промпты для генерации

## 🛡️ Безопасность

После запуска **обнови токен бота**:
1. Зайди в @BotFather
2. Команда `/revoke`
3. Выбери бота → получи новый токен
4. Обнови в Railway в переменных окружения
